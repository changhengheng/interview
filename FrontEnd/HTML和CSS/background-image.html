background-image ​

background-image：给元素设置一个或多个背景图，多个背景图会按照z轴方向层叠，最先设置的在最上层。 background-repeat：定义背景图重复方式，repeat 重复，no-repeat 不重复 background-size：背景图大小，auto：按比例缩放；cover：缩放背景图片以完全覆盖背景区，可能背景图片部分看不见；contain：缩放背景图片以完全装入背景区，可能背景区部分空白。
